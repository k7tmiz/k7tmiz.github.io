我与计算机的故事

置顶 | 发表于 2025-05-11 更新于 2026-03-13 分类于 Article

我是一个行走在文理边界上的旅人，心中始终燃烧着对计算机世界的恒久热爱。

我的故事，始于北京的家中。

父母工作繁忙，陪伴我成长的，是家里那两台早早备下的电脑。或许这算得上一种时代的馈赠，让我在懵懂的年纪便与数字世界结缘。那时的热爱简单而纯粹——查资料、玩游戏、折腾软件。至于服务器、Linux、编程这些更深的概念，还远在认知的地平线之外。

直到初中。

某一天，我在视频中第一次看见了 Linux 的世界。那种与 Windows 完全不同的系统结构与操作方式，让我产生了一种难以言说的好奇。像是无意间推开了一扇门，我开始试着迈进去看看。

我安装的第一个 Linux 发行版，是 Deepin。

大约是在 2018 年，Deepin 15 的时代。那时的 DDE 桌面还没有如今这般复杂，界面简洁，风格优雅。直到今天，我依然会偶尔关注 Deepin 的更新，比如现在的 Deepin 25。看着它一点点变化，总会生出一种奇妙的感慨。

不过 Deepin 并没有成为终点。

很快，我被 Arch Linux 的理念吸引了。

“简单、现代、实用。”

那种几乎带着一点极客气质的系统哲学，在当时的我看来格外迷人。不过第一次接触 Arch，我多少还是有些谨慎，于是选择了基于 Arch 的发行版 Manjaro + KDE 作为过渡。

也正是在那段时间，我和 KDE Plasma 结下了某种长期的缘分。

从那次与 Manjaro 的邂逅开始，KDE 就像一位默契的老朋友。无论我后来换过多少发行版，它总会在某个时刻重新出现。

至于后来究竟是因为什么最终真正投入 Arch Linux 的怀抱——

是 Arch 那极具设计感的图标？
还是当年 Manjaro 偶尔出现的“滚挂”风波？

如今已经有些记不清了。

总之，从那之后，Arch Linux 逐渐成为了我的主力系统。

那是一段相当狂热的时期。我开始不断尝试各种 Linux 发行版：

  • Kali
  • elementaryOS
  • PopOS
  • FydeOS
  • 等等等等……

至于许多发行版例如 Linux Mint，它们只在虚拟机中短暂地停留过一段时间。

如今再回头看，我已记不起我到底重装过几次系统，用过多少个发行版。

但也是因为在那个阶段，我的兴趣逐渐向更深处延伸。

我开始接触 服务器 与 编程。

初中时期，我学习了 C++。
而在 初三毕业的那个暑假，我又拥抱了 Python。

服务器系统方面，我曾经非常喜欢 CentOS 7。
后来随着它的发展路线发生变化，我逐渐转向 Debian。

从 Debian 9 到 Debian 12，我几乎都亲身经历过。

进入高中之后，生活节奏开始改变。

疫情时代的网课需求突然增加，而 Linux 对腾讯会议等工具的支持当时还不够完善。再加上学业压力逐渐加重，可以自由折腾系统的时间越来越少。

但那份热爱始终没有真正消失。

那台老旧的电脑上，依旧保持着Arch Linux 与 Windows 双系统。我仍然会在某个闲暇的夜晚，打开终端，敲下那条熟悉的命令：

有趣的是，一直到高三毕业，这个系统竟然从未真正“滚挂”。
它安静而稳定地陪伴了我整整三年。

大学开始之后，新电脑到手。

那时候电脑的主要用途变成了游戏，再加上我是文科生，于是 Windows 再次成为主角。那段时间里，我几乎很少再折腾系统。

直到某一天。

Windows 在一次关机之后并没有真正休眠，而是在背包里“空转”了整整一个月。等我再次打开电脑时，屏幕上只剩下一片花屏。

进入 PE 环境 排查之后确认是系统问题，仅仅是官方驱动的安装与调试，就花费了整整一天。

这次并不愉快的经历，加上 Windows 的强制更新、“假休眠”导致的风扇狂转，以及各种我被当作“宝宝”一样过度保护的设计——

终于再次唤醒了我对 Linux 开源，自由精神的怀念。

于是行动变得异常迅速。

我买了一块新的硬盘。

十分钟之后，Arch Linux 安装完成。

那种熟悉的感觉几乎是瞬间回来了。

不过世界也已经改变。

KDE 完成了重要的版本演进；
Steam 正在重新塑造 Linux 游戏生态；
许多新的工具与技术不断出现。

曾经积累的很多经验，已经不再完全适用。

但这并没有让我沮丧，反而重新点燃了探索的欲望。

再次深入之后，我发现整个生态早已不同：

  • 原生 Linux 版微信不再只是梦想
  • QQ 也开始认真对待 Linux 用户
  • Valve 通过 SteamDeck（搭载 Arch 系 SteamOS） 与 Proton 极大推动了 Linux 游戏生态
  • UOS 在政企领域的推广，也让国内 Linux 用户整体受益

如今，在 Linux 上畅玩游戏，早已不再是一件需要自嘲的事情。

这也让我重新燃起了体验不同发行版的兴趣：

  • openSUSE Tumbleweed 的优雅滚动
  • Ubuntu 的庞大生态
  • Fedora 的前沿特性
  • openSUSE Leap 的稳健
  • Debian 12 的纯粹
  • NixOS 那近乎革命性的理念

有趣的是，尽管我长期管理 Debian / Ubuntu 服务器，桌面端却始终没有真正深入使用过原生 Debian。

我的设备

如今，我的数字世界由两台电脑组成。

轻薄本

  • MacBook Air（主力）

游戏本

  • Arch Linux（主力系统）
  • Fedora 43
  • Windows 11（备用）

相比过去动辄四五个系统的年代，现在的设备已经经历了一轮彻底的优化。系统数量减少了，但使用体验反而更加从容。

或许，这正是折腾多年之后的一种平衡。

更新记录

本文最初完善于 2025 年 6 月 25 日。

2025 年 7 月 20 日，游戏本上的 openSUSE Tumbleweed 在一次 zypper dup 更新中因为驱动问题“壮烈牺牲”。借此机会，我也重新审视了自己的系统结构，并将系统数量精简为三个：

  • Arch Linux
  • Fedora
  • Windows 11

2025年8月13日，我换了一台轻薄本机械革命无界 14X Pro 2025 AI9 H365 32G+1T的最高配版本，遗憾的是其BIOS并不完善，没有做对Linux的支持，导致无线网卡无法正常在Linux环境下驱动。被迫在用Windows 11。这时候也把之前的洋垃圾轻薄本卖掉了。

2026年3月6日，通过国补以及以旧换新补贴，机械革命正式退役，换成了MacBook Air。

折腾仍在继续，但设备与系统组合，也逐渐走向稳定。
