# LexiForge

Universal Vocabulary TXT → JSON Converter

LexiForge is a lightweight, production-oriented TXT → JSON vocabulary converter that runs 100% in the browser (offline). Paste raw word lists copied from textbooks, PDFs, webpages, or notes, then export a clean JSON vocabulary for flashcards or your own vocabulary system.

## Highlights

- Pure frontend, offline-first: no backend, no uploads
- Text cleanup & noise filtering (empty lines, headings, separators, format notes)
- Flexible parsing: TAB-separated or space-separated lines; phrase terms supported
- Part-of-speech detection with longest-match priority
- Output modes: 📘 Wordbook JSON / 📄 Words Array
- JSON validation, copy to clipboard, download as `.json`
- Editable Table View after conversion (edit/add/delete entries, JSON updates live)
- Drag & drop import: `.txt/.tsv/.csv/.json`
- Theme modes: Auto / Light / Dark
- UI language switch: 中文 / English
- localStorage persistence (name/description/language/input/mode/theme/uiLanguage)

## Run Locally

- Open `index.html` directly in your browser
- Or serve as static files on any host (GitHub Pages, Vercel, Netlify, Cloudflare Pages)
