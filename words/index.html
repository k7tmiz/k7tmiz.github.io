用法介绍

先理解这个工具
  • 这里不是按列表一行一行刷单词，而是把单词随机排在 A4 页面上，让你按“整页内容”去记，而不是按序号机械背。
  • 你可以把“一轮”理解成一次完整学习过程。一轮默认从 1 张 A4 开始；当前页写满后，可以继续在同一轮里新增下一张。
  • 每加入 1 个新词，系统都会立刻拉你复习这一页已有的词，所以节奏不是“先攒很多再复习”，而是“新增即复习”。
第一次使用怎么开始
  • 先选择词书，然后点「下一个单词」。新词会被放到当前 A4 页面上，并马上进入自动复习。
  • 自动复习默认只针对“当前页”，适合边加边记；如果你想把这一轮所有页面一起过一遍，点「复习本轮」。
  • 复习时有三种状态：已掌握、学习中、不会。你可以点按钮，也可以直接左右滑动卡片：右滑已掌握，左滑不会。
  • 如果开启了“持续背书模式”，当前这次自动复习结束后，系统会直接开始下一次「下一个单词」，不用手动再点。
页面上几个常用入口
  • 「导入词书」支持 TXT / CSV / JSON，也支持在线导入。导入后可以直接开始学习整本词书。
  • 「查词」可以先查释义、例句或补充结果，再把词直接加入当前轮。
  • 「学习记录」里可以回看旧轮次、查看状态聚合、生成待复习轮，还能导出 CSV / PDF。
  • 「设置」里可以调整每轮上限、复习节奏、持续背书、发音、查词补充、账号与云端备份、AI 生成词书。
关于复习和翻页
  • 普通学习轮如果已经有多页，首页会出现 Previous / Next。翻页后看到的仍然是同一轮里的其他 A4 页面。
  • 轻量复习系统会根据“已掌握 / 学习中 / 不会”计算下次复习时间，记录页会自动聚合待复习内容。
  • 复习结束后弹窗会自动关闭；这是固定行为，不需要再手动确认。
发音与在线发音
  • 点击 A4 页面上的单词即可朗读。macOS / iOS 系统语音覆盖全面，Windows / Android 可能缺少非英语语音。
  • 设置中选择“在线 TTS”后，点击单词会通过网络获取发音，支持 Microsoft Edge 和 Google 翻译两种源。
  • Microsoft Edge 通常可在国内直连；Google 直连可能受当前网络环境限制，失败时会自动尝试服务端代理。
  • 在线发音会将当前朗读文本发送给所选服务；首选源不可用时可能通过服务端代理或发送给另一在线源。系统语音模式在设备本地处理。
账号与云端备份
  • 设置里的账号区默认先显示“登录”。如果还没有账号，可以切到“注册”，通过邮箱验证码完成注册。
  • 登录后才会显示云端备份按钮，可以把当前浏览器里的学习数据上传到云端，或从云端恢复回来。
  • 如果你只想本地使用，不登录也没关系，绝大多数学习功能都可以正常使用。
下载桌面 / 安卓版
  • 桌面版（macOS / Windows / Linux）和 Android 版支持自动检查更新、系统级离线使用。
  • 前往 GitHub Releases 下载最新版本。
  • Android 版下载 APK 后需允许「安装未知来源应用」，后续版本可直接覆盖安装。
键盘快捷键（电脑端）
  • Space：下一个单词
  • R：复习本轮
  • N：新建一轮
  • M：显示 / 隐藏释义
  • D：切换深浅色主题
建议第一次先直接点「下一个单词」试一轮，再去设置里调整持续背书、发音、查词和账号备份。
